## PROJECTS

### FREEGOSY
**THE ULTIMATE GAMING ECOSYSTEM.**  
A unified Flutter-based interface for RomM libraries. Designed for seamless library browsing, automated game management, and cross-platform emulator synchronization. Built for power users who demand a native, high-performance gaming dashboard.  
[SOURCE](https://github.com/abduznik/Freegosy)

---

### INSTRUMATION
**ORCHESTRATED BACKEND AUTOMATION.**  
A containerized Python orchestration layer designed for automated service instrumentation and deployment. It implements a robust web server architecture and a specialized API shim to facilitate streamlined backend workflows and high-throughput service monitoring.  
[SOURCE](https://github.com/abduznik/instrumation)

---

### TTS READER CHROME
**CONTEXTUAL SPEECH SYNTHESIS.**  
A modern extension for digital accessibility and productivity. It transforms web content into high-fidelity speech using the Web Speech API, featuring granular controls for playback dynamics and specialized voice profiles.  
[SOURCE](https://github.com/abduznik/TTS-Reader-Chrome)

---

### LDRAW2PRINT
**COMPUTATIONAL GEOMETRY FOR 3D PRINTING.**  
A technical Python utility for optimizing LDraw digital designs. It implements custom stud-welding algorithms and intelligent color separation to transform complex Lego models into production-ready physical assets.  
[SOURCE](https://github.com/abduznik/LDraw2Print)

---

### STEAMGRIDDB FOR OBSIDIAN
**DYNAMIC VISUAL LIBRARY MANAGEMENT.**  
Automated metadata and artwork orchestration for Obsidian vaults. It integrates the SteamGridDB API to fetch high-resolution grids and heroes, creating a rich, visual database for personal media collections.  
[SOURCE](https://github.com/abduznik/obsidian-steamgriddb)
